# Docker Dev Env

## How to Use??

### 1. Pull a Docker Image
```
docker pull ajtwlswjddnv/dev
```

### 2. Run
```
docker run -itd --name ajtwlswjddnv/dev dev
```

### 3. Access
```
docker exec -itd ajtwlswjddnv/dev bash
```

---

## How to Development??

### 1. Build
```
docker build -t ajtwlswjddnv/dev .
```

### 2. Push
```
docker push ajtwlswjddnv/dev
```
